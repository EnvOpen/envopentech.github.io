About Stellarae

Open Source Identity Infrastructure Built for Trust.

Stellarae is committed to building MIT-licensed security software that organizations can audit, modify, and deploy with confidence. We design for transparency first, making security controls explicit and understandable during incidents, audits, and integration.

Our Mission

Ship open source identity and authorization infrastructure that is transparent, auditable, and deployable without vendor lock-in or hidden dependencies.

How We Work

We develop openly with security reviews, comprehensive tests, operational runbooks, and documented APIs. Every component is designed to be understood and modified by your security and engineering teams.

Principles

  • Open source first. All code is MIT-licensed and auditable.
  • Service boundaries reduce blast radius without adding complexity.
  • Security controls should be explicit, observable, and under your control.
  • Documentation and operator experience are inseparable from the product.
